## Cash Machine

You need to have `composer` installed in order to run this symfony project.

```bash
$ composer update
```

## Command line

Running the code:

```bash
$ bin/console cashmachine:withdraw <amount>
```

## Unity tests

```bash
$ bin/phpunit
```
